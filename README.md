# 🎓 Student Grade Predictor

An AI-powered full-stack web application that predicts students' academic grades (A/B/C/D) using machine learning models based on lifestyle and academic habits.

---

## ✨ Features

- 🤖 Grade prediction using 3 ML models — Decision Tree, SVM & Random Forest
- 📊 Visual results with Bar & Pie charts for model comparison
- 🔐 Secure OTP-based email authentication with JWT
- 💬 AI Study Assistant chatbot powered by Groq (LLaMA 3)
- 🌙 Dark / Light mode toggle
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | React.js, Tailwind CSS, Recharts, React Router DOM |
| Auth Backend | Node.js, Express.js, MongoDB, JWT, Nodemailer |
| ML Backend | Python, FastAPI, Scikit-learn, Pandas, NumPy |
| AI Chatbot | Groq API (LLaMA 3.1) |
| Tools | Google Colab, Postman, VS Code |
